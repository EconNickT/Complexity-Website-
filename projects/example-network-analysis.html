Financial Network Analysis: Mapping S&P 500 Correlations

October 2025
3 week project
Category: Network Analysis
Python NetworkX Pandas Matplotlib Network Science
View Code Download Report

Overview

Financial markets are interconnected systems where the movement of one asset can cascade through the network, affecting many others. Understanding these interconnections is critical for risk management, portfolio construction, and identifying systemic vulnerabilities.

This project analyzes correlation networks among S&P 500 stocks to reveal hidden market structure. By applying network science techniques to stock price data, I identify key nodes, community structures, and how network topology changes during periods of market stress.

Key Motivation

Traditional correlation matrices are hard to interpret. Network visualization reveals patterns that would otherwise remain hidden in hundreds of pairwise correlations.

Methodology

Data Collection

Stock price data was collected from Yahoo Finance API for all S&P 500 constituents from January 2020 to October 2025. Daily adjusted closing prices were used to calculate returns, ensuring dividends and splits were properly accounted for.

Network Construction

The correlation network was constructed by calculating Pearson correlation coefficients between all pairs of stocks based on their daily returns. To focus on meaningful relationships, only correlations above a threshold of 0.5 were included as edges in the network.

# Calculate correlation matrix returns = price_data.pct_change().dropna() corr_matrix = returns.corr() # Build network with threshold G = nx.Graph() for i in range(len(stocks)): for j in range(i+1, len(stocks)): if corr_matrix.iloc[i, j] > 0.5: G.add_edge(stocks[i], stocks[j], weight=corr_matrix.iloc[i, j])

Network Metrics

Several network metrics were calculated to characterize the structure:

  • Degree Centrality: Identifies stocks with the most connections (highly correlated with many others)
  • Betweenness Centrality: Reveals stocks that bridge different parts of the network
  • Community Detection: Groups stocks into clusters using the Louvain algorithm
  • Network Density: Measures overall interconnectedness of the market

Results & Findings

Network Structure During Normal Conditions

Under normal market conditions, the correlation network exhibits clear sectoral clustering. Stocks within the same sector (technology, financials, healthcare) form densely connected communities, while cross-sector connections are sparser.

Figure 1: S&P 500 correlation network colored by sector. Node size represents degree centrality.

Key Findings

  • Sectoral Homophily: Stocks cluster strongly by sector, with technology and financial sectors forming the most cohesive communities.
  • Systematic Bridges: Large-cap stocks like Apple, Microsoft, and JPMorgan Chase show high betweenness centrality, acting as bridges between sectors.
  • Market Stress Behavior: During the 2022 downturn, network density increased by 37%, indicating a "correlation breakdown" where diversification benefits eroded.
  • Energy Sector Isolation: Energy stocks maintained lower correlations with other sectors, even during stress periods, suggesting potential diversification benefits.

Systemic Risk Implication

The dramatic increase in network density during market stress suggests that traditional portfolio diversification may fail precisely when it's needed most.

Centrality Analysis

The top 10 stocks by degree centrality were dominated by large-cap technology and financial stocks. These "hub" nodes represent systemic importance—their movements disproportionately affect the broader market.

Figure 2: Distribution of degree centrality across all S&P 500 stocks. Long tail indicates a scale-free network structure.

Technical Implementation

The analysis pipeline was implemented in Python using several key libraries. NetworkX provided network construction and analysis capabilities, while Pandas handled data manipulation and Matplotlib created visualizations.

Community Detection

The Louvain algorithm was used to identify communities in the network. This method optimizes modularity, finding groups of nodes with dense internal connections and sparse external connections.

from community import community_louvain # Detect communities communities = community_louvain.best_partition(G) # Calculate modularity score modularity = community_louvain.modularity(communities, G) print(f"Modularity: {modularity:.3f}") # Count communities n_communities = len(set(communities.values())) print(f"Number of communities: {n_communities}")

The analysis revealed 11 distinct communities, closely aligning with traditional sector classifications but with some interesting deviations where cross-sector correlations created hybrid groups.

Conclusions & Future Work

This project demonstrates how network science can reveal market structure that isn't apparent from traditional statistical analysis. The findings have practical implications for portfolio management, risk assessment, and understanding systemic vulnerabilities in financial markets.

Key Takeaways

  • Market structure is highly sectoral under normal conditions but becomes increasingly interconnected during stress
  • A small number of hub stocks play an outsized role in market dynamics
  • Network metrics can potentially serve as early warning indicators for market stress
  • Energy stocks maintain distinctive correlation patterns, warranting further investigation

Limitations

This analysis uses a static correlation threshold, which may miss dynamic relationships that change over time. Future work could implement rolling window analysis to capture temporal evolution of network structure. Additionally, the correlation measure captures only linear relationships and may miss more complex dependencies.

Future Directions

Several extensions could enhance this analysis:

  • Implement dynamic network analysis with time-varying correlations
  • Explore alternative similarity measures beyond Pearson correlation (e.g., mutual information, Granger causality)
  • Develop a predictive model using network metrics to forecast market volatility
  • Extend the analysis to include international markets and identify cross-border contagion pathways
  • Investigate how regulatory changes affect network structure

Practical Application

This methodology could be adapted for portfolio construction, helping identify stocks that provide true diversification benefits rather than superficial sector differences.
